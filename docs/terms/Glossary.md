# DWD Glossary

This glossary contains terms, abbreviations, and functional/context definitions used across DWD systems and processes.

These terms were extraced from the following [source](https://wigov.sharepoint.com/:x:/r/sites/DWDTEAMSO2025UIMod/_layouts/15/doc2.aspx?sourcedoc=%7BAACCAA01-1224-4AF9-AC21-2F76C010EE95%7D&file=Program%20Glossary%20-%20Reboot.xlsx&action=default&mobileredirect=true&ovuser=5a5b9e61-20b2-4578-8f37-246881fa0d61%2C20507%40nelnet.net&wdOrigin=TEAMS-MAGLEV.null_ns.rwc&wdExp=TEAMS-TREATMENT&wdhostclicktime=1769109038901&web=1)

| Term / Abbreviation | Context | Description |
|--------------------|--------|-------------|
| ACC | DWD Term | Synonym for a DWD test environment such as UAT or Test. Developers first load changes into ACC (UAT, Test) for testing and verification before moving to production. |
| ACH | External | Automated Clearing House (ACH) is a network for electronic payments and money transfers. It facilitates direct deposits, bill payments, and other secure transactions. ACH transactions are a type of Electronic Funds Transfer (EFT). |
| ACMS | DWD Term | Appeals Case Management System; part of the Mainframe. |
| ADJ | DWD Term | Adjudication or Adjudicator. |
| Agency Dashboard | Functional Area | Represents the experience and dashboard (landing page) when a DWD employee logs in. |
| Appeals | Functional Area | Functionality allowing appeals of decisions to be managed within the Employer Portal. Currently only claimants can manage appeals online; employers still use offline forms. |
| ASP | DWD Term | Adjustments and Special Processing. |
| BAM | DWD Term | Benefits Accuracy Measurement is a method of assessing the accuracy of benefits paid to claimants. |
| BD | DWD Term | Bureau Director (same persona as Office Managers). |
| Benefits Messaging / Upload | Functional Area | Capabilities for DWD and Employers to communicate and complete requests related to benefit capabilities (e.g., messaging, document upload, custom data collection). |
| Bill | DWD Term | A combination of different 'receivables' such as interest, levy amount, collection cost, solvency, reserve fund, admin fee, etc. |
| Billing & Payments | Functional Area | Work related to integrating a 3rd-party payment provider, including payment execution, storage, and reporting of past transactions. |
| BITS | DWD Department | Bureau of Information Technology Services; encompasses Mainframe, SUITES, UIBNet, and other UI systems developers. |
| BOB | DWD Term | Benefits users; Benefit Operations Bureau. |
| BOLA | DWD Term | Bureau of Legal Affairs. |
| BOSA | DWD Term | Benefits Operations Self-Assessment. |
| BTA | DWD Term | Tax and Accounting users; Bureau of Tax and Accounting. |
| CBOP | DWD Term | Benefit Overpayment Collections Unit within BTA. |
| CDEP | DWD Term | Tax Collections Unit within BTA. |
| CWC | DWD Term | Combined Wage Claim program allows an unemployed individual with wages in multiple states to combine wages to qualify for benefits under one state ("Paying state"). |
| Data Migration | Functional Area | All work related to migrating data (up to 5TB transactional databases) from existing systems to the newly developed system. |
| DBD | DWD Term | Deputy Bureau Director (same persona as Office Managers). |
| DCA | DWD Term | Denied Claims Accuracy; method for assessing the accuracy of decisions denying benefits eligibility. |
| Direct Billing Accounts | DWD Term | Also called Reimbursement Accounts. Certain employers (governmental, nonprofit, tribes) can pay unemployment claims directly instead of quarterly tax. |
| Document Upload | Functional Area | Enables employer portal users to upload documentation outside secure messaging or eForms. |
| DOR | External | Department of Revenue. |
| eForms | Functional Area | Electronic forms sent to employer accounts via the portal; can be filled out and submitted online. |
| EFT | External | Electronic Funds Transfer; digital transactions moving money between bank accounts without cash. Includes ACH, wire transfers, and electronic bill payments. |
| Employer Dashboard | Functional Area | Dashboard for employers or employer representatives showing notifications of new activity and required actions. |
| Employer Feedback | Functional Area | Previously part of the portal; now handled via external feedback form on DWD website. |
| Employer Registration | Functional Area | Allows a user to register a new employer account or update a pending registration. |
| ESS | DWD Term | Employer Services Section; manages employer accounts, investigations, tax rates, Work-Share applications, and other related issues. |
| FUTA | External | Federal Unemployment Tax Act. |
| Identity & Account Management | Functional Area | Manages authentication and authorization in Employer Portal; integrates with OKTA for user accounts and permissions. |
| Integrations | Functional Area | Integration with systems such as SUITES, UIBNET, SIDES, Appeals, and Okta as specified in the SOW. |
| IDH | DWD Term | Integrity Data Hub; secure multistate system for cross-matching UI claims and detecting fraud. |
| IQ | DWD Term | Integrity and Quality. |
| ITCH | DWD Term | "Issues That Can Happen"; tracks eligibility issues that can later be adjudicated if monetary eligibility is established. |
| LID | DWD Term | Legal Initial Determination; decision entered in Mainframe and sent to claimant/employer. |
| Mainframe | DWD System | System for processing benefits applications and appeals; integrated with SUITES, UIBNET, and claimant portal. |
| MIE | DWD Term | Manual Issue Entry Function. |
| MC | Adjudication Fact Finding | Misconduct (or discharge). |
| NSF Fees | External | Non-Sufficient Funds Fees; charged when a payment is rejected due to insufficient funds. |
| Okta | DWD System | Identity solution for authentication to the Employer Portal. Also known as MyWinconsinID |
| OS | Adjudication Fact Finding | Other Separation. |
| Other Functions | Functional Area | Miscellaneous Employer Portal functions including mailing statements, generating 940-C forms, uploading audit files, and requesting refunds or credits. |
| PCA | DWD Term | Paid Claims Accuracy; assesses accuracy of benefit payments and estimates errors. |
| PI | DWD Term | Program Integrity. |
| PPA | DWD Term | Program and Policy Analysts. |
| QC | DWD Term | Quality Control office; ensures UI money flows correctly, audits transactions, and provides reporting. |
| Qualified Settlement Fund | DWD Term | Trust/fund holding legal settlement proceeds for orderly distribution. [More info](https://dwd.wisconsin.gov/ui201/t1201.htm) |
| Receivables | DWD Term | Legally enforceable claims for payment from an employer. Includes interest, levies, collection costs, etc. |
| Reserve Fund | DWD Term | Individual account for each employer; tracks contributions, benefit payments, and determines annual tax rates. |
| S&P | DWD Term | Systems and Processing; provides business analysts and product testers for the project. |
| Secure Messaging | Functional Area | Enables secure messaging between employers and DWD staff (BTA, BOB, BOLA, QC). Supports SMS/email alerts. |
| SIDES | DWD System | Software tool for exchanging UI info electronically with employers; sponsored by USDOL. |
| Solvency Fund | External | Fund maintained from state taxes, used to pay state UI benefits. |
| SUITES | DWD System | State Unemployment Insurance Tax Enterprise System; manages tax and wage applications for Wisconsin UI. |
| SW | Adjudication Fact Finding | Suitable Work (or job refusal). |
| System Analytics & Reporting | Functional Area | Captures system metrics and analytics (drop rates, web analytics, user sessions). |
| Tax and Wage Reporting | Functional Area | Capabilities for employers to submit tax and wage reports; part of “Employer Home” and “File Report” pages. |
| Tax Messaging / Upload | Functional Area | Messaging, document upload, and data collection related to tax and wage reporting. |
| TPS | DWD Term | Tax Performance System. |
| T&O | DWD Term | Training and Outreach. |
| UIBNET | DWD System | Stores employer-claimant correspondence; system of record for related documents and tasks. [UIBNET Guide](<a href="https://wigov.sharepoint.com/:w:/r/sites/DWDTEAMUIEmployerPortalProject/Shared%20Documents/General/App%20Documentation/UIBNET/uibnet_user_guide.doc?d=wa3f777b15f3d40f3a358cd2e19d030f8&csf=1&web=1&e=JsuIHU" itemtype="http://schema.skype.com/HyperLink/Files" rel="noreferrer noopener" title="https://wigov.sharepoint.com/:w:/r/sites/dwdteamuiemployerportalproject/shared%20documents/general/app%20documentation/uibnet/uibnet_user_guide.doc?d=wa3f777b15f3d40f3a358cd2e19d030f8&csf=1&web=1&e=jsuihu" target="_blank" itemid="e9001d72-9c12-49ab-bb6e-4b1546ec84bb">https://wigov.sharepoint.com/:w:/r/sites/DWDTEAMUIEmployerPortalProject/Shared%20Documents/General/App%20Documentation/UIBNET/uibnet_user_guide.doc?d=wa3f777b15f3d40f3a358cd2e19d030f8&csf=1&web=1&e=JsuIHU</a>) |
| UIForms | DWD System | Merges Mainframe and custom info into Word docs uploaded to UIBNet. |
| VL | Adjudication Fact Finding | Voluntary Leave (or Quit). |
| WA | Adjudication Fact Finding | Work Available. |
| WFM | DWD Term | Workforce Management Analyst. |
